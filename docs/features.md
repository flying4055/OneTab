# 功能清单

## 书签管理

| 功能 | 说明 | 入口 |
|------|------|------|
| 添加书签 | 输入名称、URL、图标、背景色、文字颜色 | 加号按钮 → SiteEditorDialog |
| 编辑书签 | 修改已有书签的全部属性 | 右键菜单 → 编辑 |
| 删除书签 | 确认后删除，支持 5 秒内撤销 | 右键菜单 → 删除 → Snackbar 撤销 |
| 图标自动获取 | 输入 URL 后自动抓取 favicon（Clearbit / Google / DuckDuckGo） | 编辑器 → 自动获取 |
| 自定义图标上传 | 本地图片上传作为书签图标 | 编辑器 → 上传图片 |
| 图标背景色 | 10 色预设 + 取色器，透明图标强制白色兜底 | 编辑器 → 图标背景色 |
| 文字颜色 | 无图标时首字母显示颜色 | 编辑器 → 无图标时文字颜色 |

## 拖拽排序

| 功能 | 说明 |
|------|------|
| 鼠标拖拽 | 按住图标 ≥5px 移动触发，悬浮影子 + 原位置半透明 |
| 键盘排序 | KeyboardSensor 支持 Tab 导航 + Enter 拖拽 |
| 释放排序 | 松手后 Store 更新 → localStorage 同步写入 |
| 跨标签同步 | 一个标签页排序后，其他标签页自动刷新 |

## 搜索

| 功能 | 说明 |
|------|------|
| 搜索引擎切换 | Google / Bing / Baidu，点击左侧图标切换 |
| 本地书签过滤 | 按名称和域名模糊匹配，300ms 防抖 |
| 回车搜索 | 无匹配时直接跳转搜索引擎；有匹配时选中书签打开 |
| 自动聚焦 | 打开新标签页时光标自动定位搜索框 |

## 外观设置

| 功能 | 说明 | 默认值 |
|------|------|--------|
| 暗色主题 | 固定暗色模式，MUI 全局主题 | — |
| 自定义壁纸 | 本地上传图片，支持预览和重置 | `/bg.webp` |
| 新标签页打开 | 切换书签在当前/新标签页打开 | 当前 |
| 图标尺寸 | 滑块调节书签图标大小 | 48px (48–96) |
| 卡片间距 | 滑块调节书签横向/纵向间隔 | 16px (8–40) |
| 内容宽度 | 分阶段屏幕宽度：自动 / 400 / 640 / 768 / 1024 / 1280 / 1440 / 1680 / 1920 / 2560 | 自动 |

## 交互优化

| 功能 | 说明 |
|------|------|
| Hover 动画 | 书签卡片上浮 + 图标放大 + 阴影浮现 |
| 中键打开 | 鼠标中键点击 → 新标签页打开 |
| 右键菜单 | 在当前/新标签页打开、编辑、删除 |
| 键盘快捷键 | `Ctrl+K` 聚焦搜索、`/` 清空并聚焦、`Esc` 关闭弹窗 |
| 删除撤销 | 删除后 Snackbar 5 秒内可一键恢复 |
| 拖拽影子 | 拖拽时浮动卡片副本 + 强阴影 |
| 图片兜底 | 图标加载失败自动隐藏 → 首字母底色显示 |
| 空状态提示 | 无书签时显示"暂无书签" |

## 数据管理

| 功能 | 说明 |
|------|------|
| 导入配置 | 从 JSON 文件导入书签 + 设置 |
| 导出配置 | 导出书签 + 设置到 JSON 文件 |
| 旧版迁移 | 自动从旧 localStorage key 迁移数据 |
| 种子数据 | 首次安装加载 `nav.json` 预置书签 |

## 扩展能力

| 功能 | 说明 |
|------|------|
| MV3 兼容 | Manifest V3，支持 Chrome / Edge |
| CSP 安全 | 限制 script-src 'self'，仅允许 Google Fonts 外链 |
| favicon 权限 | 声明 `favicon` 权限 |
| 跨标签同步 | `storage` 事件监听，多标签页实时一致 |
